# Deadwax Club Site

Astro marketing and support site for Deadwax Club.

## Commands

```sh
bun install
bun run dev
bun run build
bun run preview
```

The shared list viewer expects `public/js/config.js`. For local testing, copy
`public/js/config.example.js` to `public/js/config.js` and fill in the Supabase
project values.
